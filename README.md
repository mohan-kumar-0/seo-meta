# seo-meta

Professional utility for generating SEO metadata and JSON-LD schemas for web applications.

## 🚀 Installation

```bash
npm install seo-meta
```

## 📖 Usage

```javascript
import { generatePersonSchema } from 'seo-meta';
const schema = generatePersonSchema({ name: 'John Doe' });
```
